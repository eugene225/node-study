### Node.js 모듈

- 프로그램을 작은 기능 단위로 쪼개고 파일 형태로 저장해 놓은 것
- 코드의 중복작성할 필요 없음
- 수정이 필요할 경우 필요한 모듈만 수정하면 됨

#### Node 코어 모듈

- fs : 파일시스템 관련
- HTTP : http 통신

* 글로벌 모듈 : require 없이 사용할 수 있는 모듈 ex) console.log
* 글로벌 객체 : 글로벌 모듈에 있는 객체
* __dirname / __filenmae : 현재 모듈 위치의 폴더 이름, 파일 이름을 가져옴
